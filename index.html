Html

Basicamente

HTML (HyperText Markup Language) é uma linguagem de marcação de texto que informa ao navegador como estruturar as páginas web

O HTML é formado por elementos, demarcados por tags com suas formatações

A tag <p> demarca o conteúdo e a propria tag tem sua formatação (CSS). No total temos um elemento no nosso documento HTML

Atributos

Dentro das tags é possivel adicionar caracteristicas e funções definindo atributos para as tags

Exemplo:

<p class="texto">

Ancoragem e Navegabilidade

Dentro de uma página web existem os hiperlinks, que servem para vincular nossos documentos a qualquer outro documento.
Quando colocamos o endereço no elemento especificados estamos ancorando ele com o outro elemento.
Pode acontecer dentro da própria página ou para páginas externas

Doctype

O <DOCTYPE html> é a declaração do tipo de documento. É importante definir a declaração pois garante que o documento vai ser analisado da mesma maneira por diferentes navegadores.

Exemplos:

html , xhtml , svg e etc

Lang

O <html lang> define o idioma do elemento

É usado de acordo com a ISO 639 (paises com 2 letras)

Head

O Head é a cabeça da página html, define os metadados do documento Html

Exemplos:

<title> Define o titulo da página

<base> Define um URL base que para todos os URLs relativos

<link> Define o vinculo com o documento com um recurso externo

<script> Define o vinculo com codigos executaveis ou codigos, geralmente javascript acompanhado com o "src"

<meta> Representa os metadados que não podem ser representados por outros elementos relacionados a meta HTML (charset,content,http-equiv)

Body

É o corpo do site que representa o conteúdo da página, sendo visivel para os usuários

Dentro do body, existem váris tags que estruturam o site, aqui separamos de tal maneira:

Exemplos:

Tags de Seções:

<header> <nav> <main> <article> <section> <div> <aside> <footer>

Tags de Textos

<p> <span> <pre> <b> <i> <br> <hr>

Tags de Formulários

<form> <input> <label> <button>

Tags de Tabelas

<table> <tr> <td>

Tags Multimídias

<img> <video> <audio>
